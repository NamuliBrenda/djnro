What?

DjNRO aims to provide a useful front-end for eduroam users and, most importantly, an easy to use management interface for admins of federation members (IdPs and SPs).

Where?

DjNRO is an open-source project. You can get it via github or via GRNET's DjNRO repository.

How?

We strongly recommend going through the documentation before setting up the project.

Who?

Send your comments, inquiries, remarks and ideas to our mailing list: djnro@lists.grnet.gr.

Did you know?

You can try DjNRO in action at the GRNET provided demo instance.

DjNRO powers the Greek eduroam website.

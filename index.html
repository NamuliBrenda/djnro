Welcome to DjNRO minisite

DjNRO project scope is to provide a one-stop point for eduroam users plus an administrative platform for local eduroam admins.

DjNRO is an open-source project. You can get it via github or via GRNET's DjNRO repository

Go through the documentation before setting up the project.
